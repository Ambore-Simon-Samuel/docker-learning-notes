# Docker Learning Notes

This repository contains my notes and small examples while learning Docker.

## Topics

- Docker images
- Docker containers
- Dockerfile basics
- Container networking
- Running containerised applications

## Useful Commands

Pull image
docker pull nginx

Run container
docker run -p 8080:80 nginx

List running containers
docker ps

Stop container
docker stop <container_id>

To improve my understanding of containerisation and DevOps practices as a Computing Systems & Operations student.
